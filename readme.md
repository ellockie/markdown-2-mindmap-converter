# Markdown to Mindmap converter
Converts markdown to `.mm` mindmaps
## Versions
- `main.py` - UI version; I/O only in the UI
- `main_non_ui.py` - non-UI version; I/O in dedicated files:
  - `_input_markdown.md`
  - `_output_indent.txt`

## Unit tests
Using pytest
